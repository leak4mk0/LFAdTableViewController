## Usage

To run the example project, clone the repo, and run `pod install` from the Example directory first.

## Requirements

iOS 5.0 or later

## Installation

LFAdTableViewController is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

    pod "LFAdTableViewController"
